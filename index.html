Excel Intermedio 📊

🔍

Excel Intermedio te permitirá dominar herramientas avanzadas para optimizar la organización y el análisis de datos complejos. 📈✨

En este curso aprenderás fórmulas de búsqueda y unión de texto, formato condicional para identificar visualmente tendencias, funciones lógicas y estadísticas para reportes precisos, y tablas dinámicas para sintetizar grandes volúmenes de información de manera automatizada.

💡 Atajos de teclado: Presiona Espacio para reproducir/pausar, Tab para detener

Objetivos de aprendizaje

  • 🧠 Comprender el uso de archivos CSV y la función "Texto en columnas" para importar datos correctamente.
  • 🔗 Domina la función CONCATENAR y el operador & para unir texto de múltiples celdas.
  • 🔍 Aprender a usar BUSCARV para buscar y recuperar datos de otras tablas.
  • 🎨 Aplicar formato condicional para identificar visualmente tendencias y valores importantes.
  • 📊 Utilizar funciones lógicas (SI, Y, O) y estadísticas para análisis avanzados.
  • 📈 Crear tablas dinámicas para sintetizar grandes volúmenes de información de manera automatizada.

CSV (Comma Separated Values): Son archivos de texto plano donde los datos están separados por comas. Excel los abre directamente, pero a veces es necesario usar "Texto en columnas" si no se separan automáticamente.

📝 Cómo usar Texto en columnas

  1. Selecciona la columna con los datos que deseas separar
  2. Ve a la pestaña Datos
  3. Haz clic en Texto en columnas
  4. Elige Delimitado o Ancho fijo según tus datos
  5. Selecciona el delimitador (coma, punto y coma, tabulación, etc.)
  6. Finaliza y los datos se separarán en columnas distintas

CONCATENAR: Une el contenido de dos o más celdas en una sola.

🔗 Sintaxis

  • =CONCATENAR(A1, " ", B1) - Une A1, un espacio y B1
  • =A1 & " " & B1 - Forma más corta usando el operador &

💡 Ejemplo práctico

Si tienes "Juan" en A1 y "Pérez" en B1:

=A1 & " " & B1 devuelve "Juan Pérez"

BUSCARV: Busca un valor en la primera columna de una tabla y devuelve un valor en la misma fila desde otra columna.

🔍 Sintaxis

=BUSCARV(valor_buscado, rango_tabla, num_columna, [coincidencia])

  • valor_buscado: El valor que quieres buscar
  • rango_tabla: La tabla donde buscar (debe incluir la columna de búsqueda)
  • num_columna: Número de columna desde la cual devolver el valor
  • coincidencia: VERDADERO (aproximada) o FALSO (exacta)

💡 Ejemplo práctico

Si quieres buscar el precio de un producto por su ID:

=BUSCARV(A2, Precios!A:B, 2, FALSO)

El formato condicional permite aplicar formato automáticamente a celdas que cumplen ciertas condiciones, ayudando a identificar visualmente tendencias, valores altos/bajos o datos importantes.

🎨 Tipos de reglas comunes

  • Resaltar reglas de celdas: Valores mayores que, menores que, entre, iguales a
  • Reglas superior/inferior: Los 10 primeros, 10% inferior, etc.
  • Barras de datos: Barras visuales dentro de las celdas
  • Escalas de color: Gradientes de 2 o 3 colores
  • Conjunto de iconos: Flechas, semáforos, banderas, etc.

🧠 Funciones Lógicas

  • SI: =SI(condición, valor_si_verdadero, valor_si_falso)
  • Y: =Y(condición1, condición2) - Devuelve VERDADERO si todas son verdaderas
  • O: =O(condición1, condición2) - Devuelve VERDADERO si alguna es verdadera
  • SI.ERROR: =SI.ERROR(fórmula, valor_si_error) - Maneja errores

📊 Funciones Estadísticas

  • PROMEDIO: Calcula el promedio de un rango
  • MEDIANA: Encuentra el valor central
  • MODA: Encuentra el valor más frecuente
  • DESVEST: Desviación estándar
  • CONTAR.SI: Cuenta celdas que cumplen una condición
  • SUMAR.SI: Suma celdas que cumplen una condición

Las tablas dinámicas son el recurso más potente de Excel para sintetizar grandes volúmenes de información de manera automatizada. Permiten resumir, analizar y explorar datos de forma interactiva.

📈 Cómo crear una tabla dinámica

  1. Selecciona el rango de datos que deseas analizar
  2. Ve a Insertar → Tabla dinámica
  3. Elige dónde colocar la tabla (nueva hoja o existente)
  4. Arrastra campos a las áreas: Filtros, Columnas, Filas y Valores
  5. Personaliza el formato y los cálculos según necesites

💡 Áreas de una tabla dinámica

  • Filtros: Filtra toda la tabla
  • Columnas: Categorías en la parte superior
  • Filas: Categorías en el lado izquierdo
  • Valores: Datos a calcular (suma, promedio, conteo)

Actividades prácticas

🔗 Simulación 1: CONCATENAR

Instrucciones: Selecciona la fórmula correcta para unir el nombre (A1) y apellido (B1) con un espacio en medio.

Si A1 = "Juan" y B1 = "Pérez", ¿cuál fórmula da "Juan Pérez"?

📖 Explicación:

La respuesta correcta es =A1 & " " & B1

El operador & une texto, y " " agrega un espacio entre las celdas.

🔍 Simulación 2: BUSCARV

Instrucciones: Selecciona la fórmula correcta para buscar el precio del producto con ID en A2.

Tabla de precios en hoja "Precios" (columnas A:C):

ID (A) Producto (B) Precio (C)
P001Laptop$1200
P002Mouse$25

Si A2 contiene "P001", ¿cuál fórmula devuelve el precio ($1200)?

📖 Explicación:

La respuesta correcta es =BUSCARV(A2, Precios!A:C, 3, FALSO)

  • A2: Celda con el ID a buscar
  • Precios!A:C: Rango que incluye ID, producto y precio
  • 3: Columna del precio (tercera columna)
  • FALSO: Coincidencia exacta
